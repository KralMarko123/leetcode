# LeetCode

> This repository is a small personal LeetCode workspace. I try and do all leetcode problems as C# solutions and then pair them by displaying the problems in a lightweight React app.
> The goal is to showcase my thinking process and to have a place to practice. 

- `backend/` contains the .NET solution project with the C# implementations grouped by difficulty.
- `frontend/` contains the React app that presents the problems.
- All problems are grouped by difficulty according to LeetCode's 'Easy, Medium, Hard' categories.

Check out the live site [here](https://kralmarko123.github.io/leetcode).
